Application web progressive pour apprendre la programmation, alimentée par l’intelligence artificielle. Installable sur n’importe quel appareil, sans passer par un store.
Fonctionnalités
Leçons interactives — Des cours structurés par langage avec un éditeur de code intégré et une exécution simulée par l’IA.
Défis de code — Des exercices générés par l’IA avec des cas de test automatisés, des indices progressifs et un retour immédiat sur votre solution.
Mode Bug Hunt — Retrouvez des bugs intentionnellement cachés dans du code cassé. Un chrono de 3 minutes, des récompenses en XP, et des indices disponibles à tout moment.
Classement et badges — Un classement hebdomadaire, une progression par niveaux et des badges à débloquer au fil de votre apprentissage.
Tuteur IA — Un assistant conversationnel basé sur Claude qui répond à vos questions, corrige votre code et s’adapte à votre niveau.
Mode hors-ligne — L’application est mise en cache via un service worker. Les fonctionnalités IA nécessitent une connexion internet.

Installation sur votre appareil
Android (Chrome / Edge) — Une bannière d’installation apparaît automatiquement en bas de la page après quelques secondes.
iPhone / iPad (Safari) — Ouvrez l’URL dans Safari, appuyez sur le bouton Partager, puis sur “Sur l’écran d’accueil”.
Ordinateur (Chrome / Edge) — Cliquez sur l’icône d’installation dans la barre d’adresse.

Technologies utilisées
	∙	HTML, CSS et JavaScript vanilla — aucun framework, aucune étape de build
	∙	Claude Sonnet via l’API Anthropic
	∙	Standards PWA : Web App Manifest et Service Worker API
	∙	JetBrains Mono et Syne via Google Fonts
