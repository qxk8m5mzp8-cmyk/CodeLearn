Les fonctions et la portée des variables

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique. En Python, on les définit avec le mot-clé def.

💡 Syntaxe de base
Utilisez def nom_fonction(paramètres): suivi d'un bloc indenté. Retournez une valeur avec return.

La portée (scope) détermine où une variable est accessible. Les variables définies dans une fonction sont locales à cette fonction.

Les variables définies en dehors des fonctions sont globales et accessibles partout — mais les modifier depuis une fonction nécessite le mot-clé global.

⚠️ Bonne pratique
Évitez d'abuser des variables globales. Préférez passer des valeurs en paramètres et récupérer des résultats avec return. Votre code sera plus lisible et testable.

Essayez de modifier l'exemple à droite : créez une fonction calculer_aire(largeur, hauteur) et testez-la avec différentes valeurs !
